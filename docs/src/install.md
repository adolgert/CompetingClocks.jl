# Installation

CompetingClocks.jl is a registered Julia package. You can install it using Julia's package manager.

In the Julia REPL, press `]` to enter the package manager, then run:
```julia
pkg> add CompetingClocks
```
Alternatively, you can install it directly from the REPL:
```julia
using Pkg
Pkg.add("CompetingClocks")
```
To use the package in your code:
```julia
using CompetingClocks
```
